# Sudoku
Sudoku Generator

This Java console program generates a valid solved 9 by 9 Sudoku board and prints it in a clear format.
It uses a 2D array, an ArrayList, and methods to build the board without hard-coding the final solution.

How to Run:
1. Save the file as SudokuGenerator.java.
2. Open a terminal in the project folder.
3. Compile the program SudokuGenerator.java.


Run the program:
SudokuGenerator.java


How the Board Is Generated:
The program starts with a valid Sudoku pattern, fills a 2D array, and then shuffles rows and columns within valid Sudoku groups.
Because these swaps preserve Sudoku rules, the final board remains solved.
